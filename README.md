# sql-parser

解析 mysql create table 语句，用于通过建表语句生成 model 代码文件。

```
yarn install
```

```
yarn dev
```

```
// 构建支持浏览器环境的代码文件，浏览器端直接引用构建出来的 sql-parser.min.js 文件即可，demo-browser.html 为调用示例
yarn browser
```
